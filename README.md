# Hospital Management System (SQL Server)

## Description
This mini hospital management system is developed using SQL Server.
It manages patients, doctors, appointments, and billing records.

## Features
- Patient & doctor records
- Appointment scheduling
- Billing management
- Stored procedures & triggers
- Reports using views

## Database Objects
- Tables: Patients, Doctors, Appointments, Billing
- Stored Procedures: AddAppointment, GenerateBill
- Trigger: CheckAppointmentDate
- View: PatientAppointments

## Tools
- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
